# JSON to CSV Converter

A Python script that converts JSON data into CSV format.

## Description
This project reads structured JSON data and converts it into a CSV file for easier analysis.

## Features
- JSON parsing
- CSV writing
- Handles structured data
- Useful for data conversion

## Technologies Used
- Python
- json module
- csv module

## How to Run
1. Clone the repository
2. Place JSON file in the project folder
3. Run:

   python converter.py

4. Enter JSON and CSV file names

## Example

Input:
data.json

Output:
Conversion successful.
